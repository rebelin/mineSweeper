# mineSweeper
Mine Sweeper Game Simulation for OwlHacks 2020

## Setup
Install pygame

`python3 -m pip install pygame`

The command above may not work for MAC OS, so instead try:

`python3 -m pip install -U pygame--2.0.0.dev12 --user`
